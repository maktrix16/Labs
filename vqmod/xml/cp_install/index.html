With these scripts you can either install or uninstall the database modifications needed to use te combined product mod.
It's all save, no existing tables or columns will be adjusted, only the table needed to run the modification will be added an a column will be added to your product table.

With the uninstall, these adjustments will be reversed and everything will be back the way it was.

For safety reasons, this page and it's directory should not be reachable from the internet when you don't need it. Either delete this directory or set permissions tot avoid unauthorized access.

Install combined products modification to database.

Uninstall combined products modification from database.
